## 一、处罚等级定义

### 1.1 处罚类型

| 处罚代码 | 处罚名称 | 说明 | 严重度 |
|:---|:---|:---|:---:|
| **W** | 警告 | 口头/书面警告 | 🟡 轻微 |
| **N** | 通知 | 系统通知记录 | 🟢 记录 |
| **R** | 记录 | 合规日志记录 | 🟢 记录 |
| **D** | 记过 | 正式记过一次 | 🟠 中等 |
| **C** | 绩效扣减 | 绩效分数减少 | 🟠 中等 |
| **T** | 强制培训 | 完成指定培训 | 🟠 中等 |
| **A** | 系统审计 | 触发系统级审计 | 🔴 严重 |
