## 📝 附: 批评与自我批评方法论

### 通用方法论

**原则:**
1. 实事求是 - 以事实为依据，不夸大不缩小
2. 对事不对人 - 聚焦行为和结果，不攻击人格
3. 建设性为主 - 批评是为了改进，不是为了指责
4. 及时反馈 - 问题发现后尽快提出，避免累积
5. 双向沟通 - 允许被批评者解释和申诉

**步骤:**
1. 观察: 具体描述观察到的问题行为
2. 影响: 说明该问题造成的后果
3. 感受: 表达自己的真实感受
4. 建议: 提出具体的改进建议
5. 支持: 表示愿意帮助改进

**避免:**
- 避免: '你总是...' '你从来不...'
- 避免: '我觉得你就是...'
- 避免: 在公开场合批评个人问题
- 避免: 将问题归因于性格而非行为
