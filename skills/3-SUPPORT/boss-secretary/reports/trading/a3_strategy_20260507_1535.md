## Phase 2: 情景推演

**触发条件**: RSI从89.50回落至70下方，价格跌破$80,000

**价格路径**:
```
$81,043 → $79,500 → $78,000
时间: 24-48小时内
```

**操作策略**:
- 触发: 价格跌破$80,000并企稳
- 方向: LONG试探
- 仓位: 0.01张($800@5x)
- TP: $80,500
- SL: $77,500

**验证标准**:
- ✅ 成功: 反弹至$80,500+
- ❌ 失败: 跌破$77,500

### 情景B: 区间震荡 (P=45%)

**触发条件**: 价格在$80,000-$82,800区间震荡，无明显方向

**价格路径**:
```
$80,000 ↔ $82,800
时间: 持续数天
```

**操作策略**:
- 触发: 价格触及任一边界
- 方向: 边界反向
- 仓位: 0.01张边界双向
- 网格: $80,000-$82,800，5格

**验证标准**:
- ✅ 成功: 触及边界后反向
- ❌ 失败: 突破边界并延续

### 情景C: 假突破延续 (P=15%)

**触发条件**: 价格短暂突破$82,800后快速回落

**价格路径**:
```

```

**操作策略**:
- 触发: 突破$82,800后回落至$82,000下方
