### 议题3: "价格-情绪背离"维度设计 🟡 P1

#### 背景分析

今日最显著的市场异常：
- BTC价格：$74,766 → $78,158（**+4.6%**）
- FGI：33 → 32（**-1点**）
- **背离幅度**: 价格变动+4.6%，情绪变动-3%

做梦部洞察：*"聪明资金在买，韭菜在恐惧"*

#### 背离维度定义

```
价格-情绪背离指数 (Price-Sentiment Divergence, PSD):

PSD = ΔPrice% - ΔFGI_points / 10

ΔPrice% = (当前价 - T-1价) / T-1价 × 100
ΔFGI_points = 当前FGI - T-1 FGI

示例:
  ΔPrice% = +4.6%, ΔFGI = -1
  PSD = 4.6 - (-1/10) = 4.6 + 0.1 = 4.7

阈值建议:
  |PSD| ≥ 3.0 → 强背离信号
  |PSD| ≥ 1.5 → 中背离信号
  |PSD| < 1.5 → 无背离
```

#### 背离类型分类

| PSD值 | 类型 | 解读 | 历史胜率参考 |
|:---|:---|:---|:---|
| PSD ≥ +3.0 | **价格涨+情绪不涨/跌** | 机构吸筹（聪明买，散户怕） | 买入信号偏正面 |
| PSD ≤ -3.0 | **价格跌+情绪不跌/涨** | 散户FOMO（聪明卖，散户追） | 卖出信号偏正面 |
| PSD ≈ 0 | **同向** | 正常市场 | 无额外信号 |

#### 评分权重建议

将PSD作为第8维加入评分系统（当前7维/70分制→8维/80分制）：

| 维度 | 权重 | PSD贡献规则 |
|:---|:---|:---|
| 技术面 | 15 | — |
| 宏观面 | 10 | — |
| 链上数据 | 10 | — |
| 战略匹配 | 10 | — |
| 资金费率 | 5 | — |
| FGI绝对值 | 5 | — |
| 地缘风险 | 5 | — |
| **PSD背离** | **10** | 强背离(PSD≥3)→+10分; 中背离→+5分; 无背离→0分 |

> ⚠️ 注意: PSD贡献有方向性——"价格涨+情绪不涨"对LONG加分，对SHORT减分。

#### 历史回测（概念性）

根据做梦部提供的历史案例：
- 04-19: BTC $67K→$72K(+7.5%), FGI 15→22(+7), PSD=7.5-0.7=6.8（强背离）→ 后续继续上涨
- 04-22: BTC $74.8K→$78.2K(+4.6%), FGI 33→32(-1), PSD=4.6+0.1=4.7（强背离）→ 待验证

> **初步结论**: PSD≥3.0的"价格涨+情绪不涨"场景在近期历史中偏向看涨（机构吸筹）。

### 议题4: 入场时机质量评分设计 🟡 P2

#### 背景分析

做梦部反事实损益表揭示了残酷现实：

| 情景 | 入场价 | P&L | 时机评级 |
|:---|:---|:---|:---|
| A: 停火延期买入 | $75,498 | +$2,468 | A级 |

| C: A5建议入场 | ~$77,700 | +$266 | C级 |
| **D: 实际EP46** | **$78,075** | **-$109** | **D级** |

**核心问题**: 系统只评估"方向对不对"，不评估"入场好不好"。

#### 入场时机评分维度设计

```
Timing Quality Score (TQS):

1. 相对位置评分 (30%)
   入场价相对24h高低位的位置
   TQS_pos = (24h高 - 入场价) / (24h高 - 24h低) × 100
   ≥80 → A (在区间低位买入)
   50-80 → B
   20-50 → C
   <20 → D (在区间高位追涨)

2. 回踩确认评分 (25%)
   入场前是否有回踩
   有回踩+确认 → A
   有回踩+未确认 → B
   无回踩+强信号 → C
   无回踩+弱信号 → D

3. 支撑/阻力距离评分 (25%)
   入场价距最近支撑/阻力的距离
   距支撑<1%+做多 → A (有安全垫)
   距支撑1-3% → B
   距支撑3-5% → C
   距支撑>5%或距阻力<1% → D

4. 背离信号评分 (20%)
   PSD背离状态
   PSD≥3.0+做多 → A (机构吸筹中买入)
   PSD≥3.0+做空 → D (逆机构方向)
   PSD≈0 → B
```

#### EP46回测评估

| 维度 | EP46数据 | 评分 |
|:---|:---|:---|
| 相对位置 | ($78,428-$78,075)/($78,428-$74,935)=7.5% | **D** (接近24h高点) |
| 回踩确认 | 无回踩，直接突破追入 | **D** |
| 支撑距离 | SL=$77,733，距入场$78,075仅1.6% | **C** |
| 背离信号 | PSD=4.7（强背离），做多方向正确 | **A** |
| **TQS总分** | — | **2.15/4 (D级)** |

> 📊 **结论**: EP46的方向是正确的（做多），但时机质量D级。系统如果在EP46执行前输出TQS=D的预警，A5可能选择等待回踩。

### 议题5: Demo仓位参考机制 🟡 P2

做梦部发现了一个被忽视的信号：
- Demo LONG 2张@$76,456（浮盈+$29.94）

Demo仓位显示了系统"知道"更好的价格，但实盘执行时追高了。

#### Demo参考规则设计

| 场景 | Demo信号 | 实盘动作 | 条件 |
|:---|:---|:---|:---|
| Demo盈利+实盘空仓 | 方向+价格参考 | 可参考Demo方向，但**不抄Demo价格** | Demo浮盈>1% |
| Demo盈利+实盘同向 | 仓位对比参考 | 实盘入场价应≤Demo入场价+2% | 超出则标记"追高风险" |
| Demo亏损+实盘空仓 | 反向警示 | Demo亏损方向可作为**反向信号**参考 | Demo亏损>2% |
| Demo亏损+实盘同向 | 风险加倍警示 | 双重亏损=系统性判断错误 | 立即评估是否平仓 |

#### 风险隔离原则

```
Demo参考三原则:
1. Demo≠实盘信号 — Demo仅作为"时机参考"，不作为"方向依据"
2. 参考有条件 — 仅当Demo运行>24h且未触发过SL时才参考
3. 风险隔离 — Demo仓位变化不影响实盘决策，仅作为附加信息层
```
