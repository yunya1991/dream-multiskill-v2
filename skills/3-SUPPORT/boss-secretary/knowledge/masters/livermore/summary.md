## 1. 人物背景

### 1.1 Jesse Livermore 传奇

> "华尔街没有新鲜事，因为投机像山岳一样古老。股市今天发生的，过去曾经发生过，将来也会再次发生。"

**核心数据**:
- 14岁 首次交易，盈利 $3.12
- 15岁 赚得第一个 $1,000
- 30岁 身价突破 $100万
- 42岁 巅峰时期盈利 $1亿 (相当于2024年约15亿美元)

## 2. 核心理念

### 2.1 基本信念

> "投机是一门艺术，不是科学。试图用数学来解决投机问题，就像试图用化学反应来解释爱情。"

**核心观点**:

| 观点 | 说明 |
|:---|:---|
| **时机就是一切** | 在正确的时间做正确的事 |
| **趋势是你的朋友** | 永远不要逆势而行 |
| **耐心等待** | 大钱来自大波动 |
| **认错要快** | 亏损时要跑得比兔子还快 |

## 3. 关键交易原则

### 3.1 入场时机 (The Critical Point)

Livermore 最著名的概念：**关键点 (Critical Point)**

```yaml
关键点理论:
  定义: "股价经过长期盘整后，突破重要价位的瞬间"

  识别条件:
    - 盘整时间: 通常 2-4 周
    - 突破幅度: 突破盘整区间 3-5%
    - 成交量: 突破时量能放大 2-3 倍
    - 确认: 次日收盘高于突破日

  操作:
    入场: "突破关键点时立即买入"
    止损: "关键点下方 3-5%"

  与海龟对比:
    海龟: "20日突破"
    大作手: "关键点突破 + 盘整形态确认"
    融合: "盘整后突破 + 趋势确认"
```

### 3.2 趋势确认

```yaml
大作手趋势判断:
  上升趋势:
    - "股价一底比一底高"
    - "每次回调低点高于上次"
    - "成交量在上涨时放大"

  下降趋势:
    - "股价一顶比一顶低"
    - "每次反弹高点低于上次"
    - "成交量在下跌时放大"

  横盘整理:
    - "高低点区间收窄"
    - "等待突破方向"
```

### 3.3 加仓策略

> "金字塔加仓法：只在盈利时加仓，绝不在亏损时补仓。"

```yaml
加仓规则:
  第一仓位: 30% 资金
  第二仓位: 20% 资金 (盈利后)
  第三仓位: 10% 资金 (持续盈利)
  
  总仓位上限: 60% 资金
  
  加仓条件:
    - 趋势确认延续
    - 浮盈 > 5%
    - 无反转信号
    - 仓位未超限
```

## 4. 实战技巧

### 4.1 盘整突破交易

```
案例分析: 1907 年北方太平洋做空

背景:
  - 北方太平洋铁路股价 $100-$110 盘整 3 个月
  - 利空消息: 银行业危机

错误做法 (大多数人的反应):
  - "股价这么低，可以抄底了"
  - "已经跌了这么多，不可能再跌"

大作手做法:
  - 观察盘整区间越来越窄
  - 等待向下突破 $100
  - 突破后立即做空
  - 止损设置在 $103

结果:
  - 做空后股价跌至 $50
  - 盈利超过 50%
  - 但 Livermore 这次破产了 (因为仓位过重)
```

### 4.2 关键点测试

> "股价突破后，通常会回踩关键点确认。这是最好的二次入场机会。"

```yaml
关键点回踩策略:
  形态: "突破后 3-5 天回踩突破位"
  
  确认条件:
    - 回踩低点不低于突破位 3%
    - 缩量回调
    - 次日重新上涨

  入场: "回踩确认后买入"
  止损: "突破位下方 3%"
  目标: "新高 + ATR"
```

## 5. 心理法则

### 5.1 大作手心理原则

| 法则 | 说明 |
|:---|:---|
| **不预测** | 不试图预测市场，而是跟随市场 |
| **不在意对错** | 只在意是否盈利，不在乎判断对错 |
| **快速认错** | 亏损时立即止损，不抱幻想 |
| **让利润奔跑** | 盈利时持有，等待大波动 |
| **独立思考** | 不听内幕，不随大流 |

### 5.2 常见错误

```
❌ 不要在亏损时加仓摊平成本
❌ 不要因为股价低就买入
❌ 不要因为股价高就做空
❌ 不要试图抄底或逃顶
❌ 不要忽视成交量信号
❌ 不要在横盘中频繁交易
❌ 不要情绪化交易
```

### 5.3 Livermore 的最后忠告

> "我这辈子犯过很多错误，但我从中学到了东西。一个投机者必须从自己的错误中学习，否则他会重蹈覆辙。"
