## 2. 核心理念

### 2.1 R-Multiple 系统

Tharp 最著名的创新：**用 R (Risk) 标准化所有交易**

```yaml
R-Multiple 定义:
  R = 单笔交易风险金额 (通常为账户的 1-2%)
  
  示例:
    账户: $100,000
    每笔风险: 1% = $1,000 = 1R
    
    亏损 $1,000 = -1R
    盈利 $2,000 = +2R
    盈利 $500 = +0.5R

优势:
  - 标准化所有交易
  - 跨市场可比
  - 心理压力可控
```

### 2.2 期望值公式

> "交易的本质是一个数学问题，不是预测问题。"

```yaml
期望值公式:
  E = P × R - (1-P) × L
  
  其中:
    E = 期望值
    P = 胜率
    R = 平均盈利 (R-multiple)
    L = 平均亏损 (R-multiple)

  简化版:
    E = W% × (AW/AWL) - L%
    
    W% = 胜率
    AW = 平均盈利
    AWL = 平均亏损
    L% = 败率

示例:
  胜率: 40%
  平均盈利: $2,000
  平均亏损: $1,000
  
  E = 0.4 × $2,000 - 0.6 × $1,000
  E = $800 - $600
  E = $200 (正期望值!)
```

## 3. 仓位管理系统

### 3.1 核心原则

> "在交易中，*如何做* (仓位管理) 比 *做什么* (策略) 更重要。"

```yaml
仓位管理优先级:
  1. 生存 (Survival)
     - 单笔风险 < 1-2%
     - 永远不要爆仓

  2. 稳定收益 (Steady Returns)
     - 一致性执行
     - 控制回撤

  3. 高收益 (High Returns)
     - 复利增长
     - 风险调整后最大化
```

### 3.2 固定风险仓位 (Fixed Fractional)

```yaml
固定风险仓位:
  公式: 仓位 = 账户 × 风险百分比 / (入场价 - 止损价)
  
  示例:
    账户: $100,000
    风险: 1% = $1,000
    入场价: $50
    止损价: $48
    
    每股风险: $2
    仓位: $1,000 / $2 = 500 股
    总仓位: 500 × $50 = $25,000 (25%)

关键参数:
  保守: 1% 风险
  标准: 2% 风险
  激进: 3% 风险 (需要高胜率)
```

### 3.3 固定比例仓位 (Fixed Ratio)

```yaml
固定比例法 (Tharp原创):
  公式: 新仓位 = 当前利润 / (2 × 1R)
  
  示例:
    初始 1R = $1,000
    盈利 $2,000 后:
      仓位 = $2,000 / ($2 × $1,000) = 1 单位
    
    盈利 $4,000 后:
      仓位 = $4,000 / ($2 × $1,000) = 2 单位
    
    依此类推...

优点:
  - 与利润挂钩
  - 不会因为亏损太快减少仓位
  - 鼓励坚持到盈利
```

## 4. 系统设计

### 4.1 好系统的标准

```yaml
好系统检查:
  □ 正期望值 (E > 0)
  □ 适合自己的性格
  □ 可以一致性执行
  □ 风险可管理
  □ 经过历史测试
  □ 有心理支持

正期望值示例:
  - 海龟系统: 35% 胜率, 2.5:1 盈亏比 → E ≈ 0.175R/笔
  - 突破系统: 40% 胜率, 2:1 盈亏比 → E ≈ 0.2R/笔
```

### 4.2 系统三要素

```
│                                                                      │
```

### 4.3 期望值追踪

```yaml
每月追踪表:
  月份: {月份}
  交易次数: {N}
  胜率: {W%}
  总盈亏: {总R}
  平均R/笔: {E}
  
  分解:
    最佳交易: +{X}R
    最差交易: -{Y}R

判断标准:
  E > 0.1R: 优秀
  E > 0: 合格
```

## 6. 风险控制矩阵

### 6.1 多层次风险控制

```yaml
风险层级:

  第一层: 单笔交易风险
    限制: ≤ 1-2% 账户
    方法: 固定止损

  第二层: 单日风险
    限制: ≤ 3-4% 账户
    方法: 日亏损熔断

  第三层: 单周风险
    限制: ≤ 6-8% 账户
    方法: 周亏损暂停

  第四层: 单月风险
    限制: ≤ 12-15% 账户
    方法: 月亏损复盘

  第五层: 最大回撤
    限制: ≤ 20-25% 账户
    方法: 系统性降频
```

### 6.2 风险-收益权衡

```yaml
风险收益选项:

  保守型:
    目标: 年化 20-30%
    最大回撤: < 10%
    仓位: 0.5-1% 风险
    胜率要求: > 35%

  标准型:
    目标: 年化 50-100%
    最大回撤: < 20%
    仓位: 1-2% 风险
    胜率要求: > 35%

  激进型:
    目标: 年化 200%+
    最大回撤: < 30%
    仓位: 2-3% 风险
    胜率要求: > 40%
```
